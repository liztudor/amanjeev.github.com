Book Review: CSS Mastery

CSS Mastery, it seems, is for anyone who works with CSS. It is a very helpful book for people who are new to CSS and want a well-written book that is easy to understand. I found the book extremely helpful as a thorough read and as a reference.

The author, Andy Budd, is a famous UX and Usability specialist. He has done a nice job with this book. To be honest, I actually have some permanent bookmarks that I go to when I need advice. Be it the foundations or the bugs chapter, they are worth keeping bookmarked. He starts with structuring your code, teaching better semantics and strategies. He also covers various techniques which are important as a CSS developer like FIR. I also like that he covered formatting links, lists and forms. At the end of the book, there are two great case-studies which are very helpful for someone who wants to begin front-end web development.

I just have one gripe which is not a big anyway. I hope that he had a separate chapter or section for CSS3 standard rather than it being within each chapter. A separate section would have been awesome for people who are trying to get on the CSS3 bandwagon. In short, this is a great book for a day-to-day CSS developer to keep by his/her side.
